# 🛒 Amazon Sales Analysis using Python

## 📌 Project Overview

This project focuses on analyzing Amazon sales data using Python Data Analytics techniques to identify sales trends, customer purchasing behavior, store performance, and business insights.

The project demonstrates the complete Data Analytics workflow including:

✅ Data Collection

✅ Data Cleaning

✅ Data Preprocessing

✅ Exploratory Data Analysis (EDA)

✅ Data Visualization

✅ Business Insights Generation

✅ Data-Driven Decision Making

---

## 🎯 Project Objective

The objective of this project is to analyze Amazon sales data and discover meaningful patterns that can help businesses improve sales performance, optimize inventory management, understand customer behavior, and support strategic decision-making.

---

## 🗂️ Dataset Information

The dataset contains Amazon sales-related information including:

- 🏪 Store Information
- 🛍️ Department Details
- 💰 Weekly Sales
- 🏷️ Store Type
- 🌡️ Temperature
- ⛽ Fuel Price
- 📈 CPI (Consumer Price Index)
- 👨‍💼 Unemployment Rate
- 🎉 Holiday Information
- 💸 Markdown Discounts
